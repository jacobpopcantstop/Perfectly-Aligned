The creative drawing game for ethically dubious people!

View on GitHub Host Your Own

How It Works

1

Host a Room

One player opens the host screen on a TV or projector and creates a room with a 4-letter code.

2

Players Join

Everyone else joins on their phones or tablets by entering the room code.

3

Roll an Alignment

The rotating Judge rolls a D&D alignment and picks a prompt card for the round.

4

Draw & Judge

Everyone draws their interpretation. The Judge picks a winner and awards points!

Features

Jackbox-Style Play

One screen hosts the game while everyone joins on their own device. No app install needed.

Multiple Card Decks

Core Mix, Creative & Pop Culture, and Hypothetical & Taboo (17+) decks to choose from.

Token System

Earn tokens for special achievements. Spend them to re-roll prompts or steal points.

Real-Time Drawing

HTML5 Canvas drawing on any device with instant submission via WebSockets.

Host Your Own Game

Perfectly Aligned is a real-time multiplayer game that requires a Node.js server. Follow these steps to run it:

  1. Clone the repository 
    git clone https://github.com/jacobpopcantstop/Perfectly-Aligned.git
cd Perfectly-Aligned
  2. Install dependencies 
    npm install
  3. Start the server 
    npm start
  4. Play! Open http://localhost:3000 on your main screen and share the room code with players.

For online play, deploy to any Node.js hosting platform such as Render, Railway, or Glitch.
